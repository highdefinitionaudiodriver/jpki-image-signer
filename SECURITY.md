# Security Policy

## サポートバージョン

| Version | Supported |
|---|---|
| 0.1.x   | ✅ |
| < 0.1   | ❌ |

## 脆弱性の報告

セキュリティ上の問題を発見した場合は、**Issue や Pull Request での公開報告は避け**、以下のメールアドレスへ直接ご連絡ください。

- **連絡先**：highdefinitionaudiodriver@gmail.com
- **対応 SLA**：ベストエフォート（個人開発のため即時対応はお約束できませんが、72時間以内に初動連絡を目指します）

報告に含めていただけると助かる情報：

- 影響を受けるバージョン（`v0.1.0` 等）
- 再現手順
- 想定される影響範囲（PII 漏洩・改ざん検知バイパス・PIN 漏洩 等）
- 可能であれば修正案

## 本ツール固有のセキュリティ事項

- **PIN の取り扱い**：PIN は受信後即座に `bytearray` 化し、APDU 送信直後にゼロクリアします。ログ・例外メッセージに PIN を含めません。
- **PIN ロック対策**：残回数 < 3 で送信前に中止します。実機テストの際は十分注意してください。
- **`.jpkiimg` には署名用電子証明書（氏名・住所・生年月日等を含む）が同梱されます**。公開リポジトリ・SNS への投稿は避けてください。
- **証明書チェーン検証 / 失効確認は本バージョン未実装**です。改ざん検知は可能ですが、証明書の有効性確認は別途必要です。
